# Input Key Glossary

- `day_summary`: completed-day operational facts.
- `shift_policy`: strategist intent and deterministic compiled policy summary.
- `reviewer_report`: reviewer diagnosis and next-day correction signals.
- `existing_wiki_digest`: compact prompt-facing wiki context.
- `existing_graph_digest`: compact prompt-facing graph context.
- `schema_views.managed_objects`: object lenses: items, workers, equipment, queues.
- `schema_views.manager_perspectives`: manager lenses: strategist, reviewer, curator.
