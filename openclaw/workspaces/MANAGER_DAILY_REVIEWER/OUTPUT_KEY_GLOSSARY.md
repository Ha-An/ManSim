# Output Key Glossary

## `target_misses`
List of missed daily target keys only.

## `top_failure_modes`
Canonical failure labels only.

## `recommended_prevention_targets`
Up to 2 canonical prevention targets for tomorrow.

## `recommended_support_pair`
Exactly one support pair such as `A1->A3`.

## `role_change_advice`
Sparse role hints only when tomorrow should clearly keep or restore a canonical role.

## `carry_forward_risks`
Short English risk strings that tomorrow's strategist should keep in mind.
