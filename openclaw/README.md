# OpenClaw 템플릿

이 디렉터리는 ManSim이 사용하는 OpenClaw 프로파일과 워크스페이스 템플릿을 저장합니다.

## 구성
- `profiles/mansim_repo`: OpenClaw 프로파일 설정
- `workspaces/A1`, `A2`, `A3`: 작업자 템플릿
- `workspaces/MANAGER`: 공통 매니저 템플릿
- `workspaces/MANAGER_BOTTLENECK_DETECTOR`: 병목 탐지기 템플릿
- `workspaces/MANAGER_DAILY_PLANNER`: 일일 계획기 템플릿

## 템플릿 파일 의미
- `SOUL.md`: 역할 성향과 판단 자세
- `IDENTITY.md`: 책임 경계와 전문성
- `AGENTS.md`: 현재 워크스페이스에서 지켜야 할 규칙
- `TOOLS.md`: 사용 가능한 도구와 사용 원칙
- `BOOTSTRAP.md`: 시작 시 읽을 최소 지침
- `HEARTBEAT.md`: 변하지 않는 운영 원칙
- `MEMORY.md`: 런타임 요약 메모리 파일
- `USER.md`: 현재 턴 지시를 쓰는 scratch 파일
